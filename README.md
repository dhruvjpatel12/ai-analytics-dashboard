# 🚀 AI Analytics Dashboard

Transform any Excel dataset into AI-powered business insights in seconds.

An intelligent Business Intelligence dashboard built using **n8n**, **Google Gemini AI**, and **dynamic HTML reports**. The platform automatically analyzes uploaded Excel datasets, generates KPIs, creates interactive charts, and produces AI-powered executive summaries.

---

## ✨ Features

- 📊 Automatic Excel data analysis
- 🤖 AI-generated executive summaries using Google Gemini
- 📈 Dynamic KPI generation
- 📉 Automatic chart creation based on dataset
- 🧹 Data cleaning and validation
- 📋 Interactive HTML dashboard
- 📄 Export dashboard to PDF
- 🔄 Refresh dashboard instantly
- 🧠 Schema-independent analytics engine
- 📦 Works with multiple dataset types

## 🚀 Getting Started

1. Import `workflow_dashboard.json` into n8n.
2. Configure your Google Gemini API credentials.
3. Execute the workflow.
4. Open `index.html`.
5. Upload an Excel dataset.
6. Explore the generated dashboard.
